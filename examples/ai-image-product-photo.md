# AI Image Product Photo Example

## Scenario

Use this workflow to write a clean AI image prompt for a product photo while preserving the real product shape and details.

## Example Input

```text
Product: polarized sunglasses
Audience: outdoor sports users
Platform: product listing
Language: English
Tone: clean commercial
```

## Prompt Used

Use the standardized [AI Image Product Photo](../docs/STANDARDIZED_PROMPTS.md#ai-image-product-photo) prompt.

## Expected Output

A copy-ready image prompt including:

- Subject
- Product details
- Background
- Composition
- Lighting
- Camera style
- Negative constraints

## Notes

Do not change the product logo, shape, color, or visible structure. If text accuracy matters, add text later in a design tool.
