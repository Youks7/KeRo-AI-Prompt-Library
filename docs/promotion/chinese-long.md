我整理了一个开源项目：KeRo AI Prompt Library。

这是一个面向电商、TikTok、AI 生图、AI 视频、视频分镜、文案生成、本地化翻译和自动化工作流的实用 Prompt 工作流库。

它不是简单堆提示词，而是按照真实内容生产场景整理：

- 电商产品文案
- TikTok 视频脚本
- AI 生图提示词
- AI 视频分镜
- 产品海报提示词
- 翻译和本地化
- 自动化内容生产工作流

适合跨境电商运营、内容创作者、独立开发者和 AI 工具使用者。

如果你正在用 ChatGPT、Claude、Gemini、Midjourney、Seedance、Runway 等工具做内容生产，可以直接复制里面的 Prompt 使用，也可以点击 Use this template 创建自己的 Prompt 库。
