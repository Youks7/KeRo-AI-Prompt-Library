# Translation and Localization Example

## Scenario

Use this workflow to localize product or short-video copy into a natural target language.

## Example Input

```text
Product: polarized sunglasses
Audience: outdoor sports users
Platform: TikTok
Language: Thai
Tone: friendly and practical
```

## Prompt Used

Use the standardized [Translation and Localization](../docs/STANDARDIZED_PROMPTS.md#translation-and-localization) prompt.

## Expected Output

- Localized version
- Literal meaning notes when useful
- Cultural or platform-risk notes
- Alternative tone version

## Notes

For customer-facing Thai copy, have a native speaker review the final version when possible.
