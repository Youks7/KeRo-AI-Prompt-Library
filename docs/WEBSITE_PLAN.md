# Website Plan

## Goal

Turn KeRo AI Prompt Library into a simple, searchable, user-friendly prompt website.

The GitHub repository remains the source of truth, while the website becomes the public-facing product entrance.

## Recommended Domains

- keroai.org
- keroai.xyz
- keroai.co

## Recommended Deployment

- Cloudflare Pages
- GitHub Pages
- Vercel

## Recommended Structure

- `/` Home
- `/prompts` All prompts
- `/prompts/ecommerce`
- `/prompts/tiktok`
- `/prompts/image-generation`
- `/prompts/video`
- `/packs`
- `/examples`
- `/about`
- `/github`

## Homepage Sections

1. Hero section
2. Project introduction
3. Prompt categories
4. Featured prompts
5. Example workflows
6. GitHub link
7. GitHub repository link

## Prompt Card Fields

Each prompt card should include:

- Title
- Category
- Use case
- Input variables
- Prompt body
- Copy button
- Example input
- Expected output
- Notes

## Future Features

- Search
- Category filter
- Copy button
- Language switch
- Prompt detail pages
- Example image gallery
- Prompt pack download
- GitHub Star button
