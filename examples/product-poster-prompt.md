# Product Poster Prompt Example

## Scenario

Use this workflow to create a commercial poster prompt for product promotion or social content.

## Example Input

```text
Product: polarized sunglasses
Audience: outdoor sports users
Platform: Instagram / TikTok
Language: English
Tone: modern and clean
```

## Prompt Used

Use the standardized [Product Poster Prompt](../docs/STANDARDIZED_PROMPTS.md#product-poster-prompt) prompt.

## Expected Output

- Short product and audience analysis
- Layout recommendation
- Final poster prompt
- Negative prompt

## Notes

Avoid fake user numbers, fake star counts, fake awards, and excessive marketing text.
