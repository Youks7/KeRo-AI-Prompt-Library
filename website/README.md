# KeRo AI Prompt Library Website

This folder contains a minimal static website for KeRo AI Prompt Library.

## Files

- `index.html`: homepage markup
- `styles.css`: responsive styling

## Deployment

This first version does not use a backend, database, or complex framework. It can be deployed with:

- Cloudflare Pages
- GitHub Pages
- Vercel static hosting

For Cloudflare Pages or GitHub Pages, set the project root to `website/` when deploying this static version.
