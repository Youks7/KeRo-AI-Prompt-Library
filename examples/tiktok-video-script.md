# TikTok Video Script Example

## Scenario

Use this workflow to create a short product video plan with a strong first-three-second hook and clear shot structure.

## Example Input

```text
Product: polarized sunglasses
Audience: outdoor sports users
Platform: TikTok
Language: English
Tone: persuasive
```

## Prompt Used

Use the standardized [TikTok Short Video Script](../docs/STANDARDIZED_PROMPTS.md#tiktok-short-video-script) prompt.

## Expected Output

- Audience psychology analysis
- Three title or caption options
- Shot-by-shot script table
- Three alternative hooks
- CTA suggestion

## Notes

Keep the product placement natural and avoid fake scarcity, fake discounts, or unsupported performance claims.
