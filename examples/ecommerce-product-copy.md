# E-commerce Product Copy Example

## Scenario

Use this workflow to turn product facts into detail-page copy without inventing unsupported claims.

## Example Input

```text
Product: polarized sunglasses
Audience: outdoor sports users
Platform: TikTok Shop
Language: English
Tone: persuasive
```

## Prompt Used

Use the standardized [E-commerce Product Detail Copy](../docs/STANDARDIZED_PROMPTS.md#e-commerce-product-detail-copy) prompt.

## Expected Output

- Main selling point
- Pain-point and solution pairs
- Short selling points
- Specification copy based only on provided facts
- Usage scenarios
- FAQ
- Soft and conversion-focused CTA

## Notes

Verify polarization, UV protection, material, price, and certifications before publishing.
