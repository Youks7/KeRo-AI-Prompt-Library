## Overview

KeRo AI Prompt Library is a practical collection of structured AI workflows maintained and tested by Kero. Instead of isolated one-line prompts, each template combines a role, task, inputs, constraints, output format, and iteration method.

The current release contains **7 categories, 40+ practical prompts, and 8 advanced prompt packs** for e-commerce, short-form video, TikTok, Chinese–Thai localization, AI image/video creation, and market-report automation.

> Most prompt bodies are written in Simplified Chinese, with selected English and Thai content. This English README provides a clear project map for international readers.

## How to use

1. Choose the category closest to your task.
2. Expand an item and copy its code block.
3. Replace variables such as `{product}`, `{audience}`, `{platform}`, and `{goal}`.
4. Provide verified product data and references.
5. Review and iterate before publishing.

## Responsible use

- Verify product claims, financial data, platform rules, and model capabilities before use.
- Only upload or adapt media, fonts, brands, and likenesses you have the right to use.
- Review every output for accuracy, copyright, privacy, safety, and platform compliance.
